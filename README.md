## Guide

1. Install packages
    ```
    yarn
    ```
    or
    ```
    npm install
    ```

2. Start the app
    ```
        yarn start
    ```
   or
   ```
       npm run start
   ```
   
3. If you want to run the unit tests

    ```
        yarn test
    ```
   or
   ```
       npm run test
   ```

## Note   
In order to enable cheat mode, head to `.env` file, replace 0 with 1 and restart the app
